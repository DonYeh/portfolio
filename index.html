Chatris

FULL-STACK MOBILE APP

Chatris is a cross-platform, full-stack mobile app that allows you to enjoy a timeless classic while connecting with your friends over video and voice chat. Chatris is built using React Native, PostgreSQL, Node, Express, and Socket.io

Movie-Explorer

Front-End web app

Movie Explorer allows you to browse and discover movies. It uses a responsive, mobile-first design featuring advanced filtering and search functionality. It is built with Javascript, HTML, CSS and it calls the OMDB and TMDB APIs for up-to-date movie information and trailers.

Learn Code Share

BACK-END WEB APP

Learn-Share-Code is a knowledge-base designed for newer developers to use as an educational tool. Users can catch up on news in the tech industry, review methods from multiple languages, and submit new ones for the community. It is built with HTML, Javascript, Node.js, Express, and PostgreSQL. Learn-Share-Code incorporates the Model-View-Controller (MVC) architecture design pattern.
